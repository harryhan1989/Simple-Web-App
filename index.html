Simplewebapp

This is a simple web app starter project with authentication, users, roles, and role management using NodeJS, Jade, and MongoDb

About SimpleWebApp

Developed by @tshrove to help developers with starting a simple web application using NodeJS, Jade, and MongoDB. It contains:

  • User Management
  • User Roles
  • Authentication
  • Register and Login Page
  • Admin Page
  • Starter Homepage
